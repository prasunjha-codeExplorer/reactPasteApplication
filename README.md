# 🚀 React Paste Application

A simple and efficient paste management app built using React. This application allows users to create, store, search, and manage text snippets (pastes) with a clean and user-friendly interface.

## 🌐 Live Demo

👉 https://react-paste-application.vercel.app/

---

## ✨ Features

* 📝 Create and save text pastes
* 🔍 Search pastes easily
* 📋 Copy pastes to clipboard
* 🗑️ Delete unwanted pastes
* 💾 Persistent storage using LocalStorage
* ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

* React.js
* Redux Toolkit
* Tailwind CSS
* Vite

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/react-paste-application.git
```

2. Navigate to the project folder:

```
cd react-paste-application
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```
